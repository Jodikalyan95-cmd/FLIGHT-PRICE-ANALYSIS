# FLIGHT-PRICE-ANALYSIS
An interactive Power BI dashboard built using flight price data to analyze airline pricing, flight durations, stops, travel patterns, and city-to-city trends. Includes comparisons by airline, class, arrival time, source/destination city, and more. Perfect for data analytics portfolio projects.
